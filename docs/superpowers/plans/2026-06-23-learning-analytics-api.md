# Learning Analytics API (H1) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add personal learning analytics read APIs (FR-ANA-01..07) plus an `error_type` self-classification on `UserQuestionState`, all computed from existing practice/exam answer tables.

**Architecture:** New `analytics` bounded concern = service (`app/services/analytics.py`) + router (`app/api/analytics.py`) + schemas (`app/schemas/analytics.py`), mirroring sub-projects E/F. One schema change: nullable `error_type` enum column on `UserQuestionState`, exposed through the existing `PUT /api/practice/questions/{id}/state`. Aggregations run two grouped queries (practice + exam answers) per call and merge in Python; all queries are personal-scoped (`user_id == current.user.id`).

**Tech Stack:** FastAPI, SQLAlchemy 2.x, Alembic, Pydantic v2, PostgreSQL 16 (native ENUM, JSONB), real `cissp_test` DB for tests.

## Global Constraints

- Tests use the real PostgreSQL `cissp_test` DB (NOT dev `cissp`), per-test SAVEPOINT rollback, `Base.metadata.create_all` (NOT migrations). Migration-drift test runs against `cissp_migtest`.
- Service-layer backend: route handlers delegate to `app/services/analytics.py`; no business logic in routes. Read-only endpoints do NOT commit. The one mutation (error-type) reuses `set_question_state`, which the existing route already commits.
- Tenant/ownership scoping: every analytics query filters `user_id == current.user.id`. Domain/knowledge-point taxonomy reads are GLOBAL. The classification write keeps `set_question_state`'s existing `organization_id` guard.
- Permission gating: all analytics endpoints gated by `require_permission("practice:read")`. Do NOT add a new permission code (no seed/matrix change).
- Native PostgreSQL ENUM: `error_type` is a real `CREATE TYPE` in the migration; the migration's `downgrade()` must `DROP TYPE error_type` explicitly (autogen `drop_table` does not).
- Soft delete only via `not_deleted(model)` where questions are read. Historical integrity: never read live `Question` rows for answer data — use `PracticeAnswer`/`ExamAnswer` rows + their snapshots.
- No business logic in routes; ORM/parameterized queries only (no raw string SQL).
- Do NOT name a route handler `get_session` (shadows the DB-session dependency import).
- Analytics degrades gracefully: empty/new user → 200 with zero/empty structures (never 404); no current blueprint → domain-mastery returns `[]`, recommendation returns `focus_domain=None` + empty lists, 200 (NOT 422).
- `error_type` enum values (exact strings): `concept_unclear`, `misread_stem`, `memory_lapse`, `option_confusion`, `time_pressure`.
- Mastery-from-accuracy thresholds (exact): `>=0.8 mastered`, `>=0.6 reviewing`, `>=0.4 learning`, else `not_started` (matches `MasteryLevel` enum values).
- Weak-area threshold (exact): `accuracy < 0.6 AND answered >= 3`; weak_domains sorted accuracy asc, top 8; weak_knowledge_points top 10.
- `window_days` query param: int, must be 30 or 90; default 30; 422 otherwise.
- `next_practice_question_ids`: `<=10`, from weak areas, least-recently-practiced (earliest `answered_at`), excluding already-mastered questions.
- The uncommitted working-tree edit to `docs/CISSP_EXAM_PRACTICE_SYSTEM_PRD.md` is NOT ours — never stage/commit/clobber it. Stage only intended files.

Create `backend/app/services/analytics.py`:

```python
from __future__ import annotations
from collections import defaultdict
from datetime import date, datetime, timedelta, timezone
from sqlalchemy import select, func
from sqlalchemy.orm import Session

from app.models.practice import PracticeAnswer, UserQuestionState
from app.models.exam import ExamAnswer
from app.models.question import QuestionMapping
from app.models.taxonomy import ExamBlueprint, ExamDomain
from app.models.enums import MasteryLevel
from app.schemas.analytics import (
    DashboardOut, DomainMasteryOut, TrendPoint, TrendOut,
)


def _current_blueprint_or_none(session: Session) -> ExamBlueprint | None:
    return session.execute(
        select(ExamBlueprint).where(ExamBlueprint.is_current.is_(True))
    ).scalars().first()


def _mastery_from_accuracy(acc: float) -> MasteryLevel:
    if acc >= 0.8:
        return MasteryLevel.mastered
    if acc >= 0.6:
        return MasteryLevel.reviewing
    if acc >= 0.4:
        return MasteryLevel.learning
    return MasteryLevel.not_started


def _streak(dates: set[date]) -> int:
    if not dates:
        return 0
    today = datetime.now(timezone.utc).date()
    streak = 0
    day = today
    while day in dates:
        streak += 1
        day -= timedelta(days=1)
    return streak


def _answer_rows(session: Session, user_id, since=None):
    """Yield (question_id, is_correct, time_spent_ms, answered_at) for practice+exam."""
    pa = select(
        PracticeAnswer.question_id, PracticeAnswer.is_correct,
        PracticeAnswer.time_spent_ms, PracticeAnswer.answered_at,
    ).where(PracticeAnswer.user_id == user_id)
    ea = select(
        ExamAnswer.question_id, ExamAnswer.is_correct,
        ExamAnswer.time_spent_ms, ExamAnswer.answered_at,
    ).where(ExamAnswer.user_id == user_id)
    if since is not None:
        pa = pa.where(PracticeAnswer.answered_at >= since)
        ea = ea.where(ExamAnswer.answered_at >= since)
    out = []
    out.extend(session.execute(pa).all())
    out.extend(session.execute(ea).all())
    return out


def _answer_buckets(session: Session, user_id, since=None):
    """Return dict[domain_id] -> {answered, correct, time_ms} merged across practice+exam."""
    rows = _answer_rows(session, user_id, since=since)
    qids = {r[0] for r in rows}
    if not qids:
        return {}
    dom_map = dict(session.execute(
        select(QuestionMapping.question_id, QuestionMapping.domain_id)
        .where(QuestionMapping.question_id.in_(qids))
    ).all())
    buckets: dict = defaultdict(lambda: {"answered": 0, "correct": 0, "time_ms": 0})
    for qid, is_correct, tms, _answered_at in rows:
        dom = dom_map.get(qid)
        if dom is None:
            continue
        b = buckets[dom]
        b["answered"] += 1
        if is_correct:
            b["correct"] += 1
        b["time_ms"] += tms or 0
    return buckets


def dashboard(session: Session, *, user_id) -> DashboardOut:
    rows = _answer_rows(session, user_id)
    total = len(rows)
    correct = sum(1 for r in rows if r[1])
    study = sum((r[2] or 0) for r in rows)
    last = max((r[3] for r in rows), default=None)
    days = {r[3].astimezone(timezone.utc).date() for r in rows}
    return DashboardOut(
        practiced_questions=len({r[0] for r in rows}),
        total_answered=total,
        correct_count=correct,
        accuracy=round(correct / total, 4) if total else 0.0,
        study_time_ms=study,
        streak_days=_streak(days),
        last_active_at=last,
    )


def domain_mastery(session: Session, *, user_id, blueprint) -> list[DomainMasteryOut]:
    if blueprint is None:
        return []
    domains = session.execute(
        select(ExamDomain).where(ExamDomain.blueprint_id == blueprint.id)
        .order_by(ExamDomain.number)
    ).scalars().all()
    buckets = _answer_buckets(session, user_id)
    out = []
    for d in domains:
        b = buckets.get(d.id, {"answered": 0, "correct": 0, "time_ms": 0})
        ans = b["answered"]
        cor = b["correct"]
        acc = round(cor / ans, 4) if ans else 0.0
        out.append(DomainMasteryOut(
            domain_id=d.id, number=d.number, name=d.name, weight_pct=d.weight_pct,
            answered=ans, correct=cor, accuracy=acc,
            avg_time_ms=round(b["time_ms"] / ans) if ans else 0,
            mastery_level=_mastery_from_accuracy(acc).value,
        ))
    return out


def trend(session: Session, *, user_id, window_days: int) -> TrendOut:
    if window_days not in (30, 90):
        raise ValueError("window_days must be 30 or 90")
    since = datetime.now(timezone.utc) - timedelta(days=window_days)
    rows = _answer_rows(session, user_id, since=since)
    by_day: dict[date, dict] = defaultdict(lambda: {"answered": 0, "correct": 0})
    for _qid, is_correct, _tms, answered_at in rows:
        day = answered_at.astimezone(timezone.utc).date()
        by_day[day]["answered"] += 1
        if is_correct:
            by_day[day]["correct"] += 1
    points = []
    for day in sorted(by_day):
        b = by_day[day]
        points.append(TrendPoint(
            date=day, answered=b["answered"], correct=b["correct"],
            accuracy=round(b["correct"] / b["answered"], 4),
        ))
    return TrendOut(window_days=window_days, points=points)
```

def weak_areas(session: Session, *, user_id) -> WeakAreasOut:
    buckets = _answer_buckets(session, user_id)
    # domains
    dom_rows = session.execute(
        select(ExamDomain.id, ExamDomain.name)
    ).all() if buckets else []
    dom_name = {r[0]: r[1] for r in dom_rows}
    weak_d = []
    for dom_id, b in buckets.items():
        ans = b["answered"]
        if ans < _WEAK_MIN_ANSWERED:
            continue
        acc = b["correct"] / ans
        if acc < _WEAK_ACC:
            weak_d.append(WeakAreaOut(
                domain_id=dom_id, knowledge_point_id=None,
                label=dom_name.get(dom_id, str(dom_id)),
                answered=ans, correct=b["correct"], accuracy=round(acc, 4),
            ))
    weak_d.sort(key=lambda w: w.accuracy)
    # knowledge points
    rows = _answer_rows(session, user_id)
    qids = {r[0] for r in rows}
    kp_map = dict(session.execute(
        select(QuestionMapping.question_id, QuestionMapping.knowledge_point_id)
        .where(QuestionMapping.question_id.in_(qids), QuestionMapping.knowledge_point_id.is_not(None))
    ).all()) if qids else {}
    kp_ids = {v for v in kp_map.values() if v is not None}
    kp_name = {r[0]: r[1] for r in session.execute(
        select(KnowledgePoint.id, KnowledgePoint.name).where(KnowledgePoint.id.in_(kp_ids))
    ).all()} if kp_ids else {}
    kp_buckets: dict = defaultdict(lambda: {"answered": 0, "correct": 0})
    for qid, is_correct, _tms, _at in rows:
        kp = kp_map.get(qid)
        if kp is None:
            continue
        b = kp_buckets[kp]
        b["answered"] += 1
        if is_correct:
            b["correct"] += 1
    weak_k = []
    for kp_id, b in kp_buckets.items():
        ans = b["answered"]
        if ans < _WEAK_MIN_ANSWERED:
            continue
        acc = b["correct"] / ans
        if acc < _WEAK_ACC:
            weak_k.append(WeakAreaOut(
                domain_id=None, knowledge_point_id=kp_id,
                label=kp_name.get(kp_id, str(kp_id)),
                answered=ans, correct=b["correct"], accuracy=round(acc, 4),
            ))
    weak_k.sort(key=lambda w: w.accuracy)
    return WeakAreasOut(weak_domains=weak_d[:_MAX_WEAK_DOMAINS], weak_knowledge_points=weak_k[:_MAX_WEAK_KPS])


def error_type_breakdown(session: Session, *, user_id) -> ErrorTypeOut:
    # wrong answers = rows where is_correct is False
    rows = _answer_rows(session, user_id)
    wrong_qids = {r[0] for r in rows if r[1] is False}
    et_map = dict(session.execute(
        select(UserQuestionState.question_id, UserQuestionState.error_type)
        .where(UserQuestionState.user_id == user_id, UserQuestionState.question_id.in_(wrong_qids))
    ).all()) if wrong_qids else {}
    counts: dict = defaultdict(int)
    classified = 0
    for qid in wrong_qids:
        et = et_map.get(qid)
        counts[et.value if et is not None else None] += 1
        if et is not None:
            classified += 1
    # ordered: the 5 enum types then None
    order = [e.value for e in ErrorType] + [None]
    distribution = [ErrorTypeBreakdown(error_type=k, count=counts.get(k, 0)) for k in order if counts.get(k, 0) > 0 or k is None]
    return ErrorTypeOut(total_wrong_classified=classified, distribution=distribution)


def recommendation(session: Session, *, user_id, blueprint) -> ReviewRecommendationOut:
    if blueprint is None:
        return ReviewRecommendationOut(
            focus_domain=None, wrong_to_review=[], next_practice_question_ids=[],
            rationale="No current exam blueprint configured.",
        )
    wa = weak_areas(session, user_id=user_id)
    focus = wa.weak_domains[0] if wa.weak_domains else None
    # wrong-to-review: wrong questions not mastered, not flagged-resolved
    rows = _answer_rows(session, user_id)
    wrong_qids = [r[0] for r in rows if r[1] is False]
    states = session.execute(
        select(UserQuestionState).where(
            UserQuestionState.user_id == user_id,
            UserQuestionState.question_id.in_(wrong_qids),
        )
    ).scalars().all() if wrong_qids else []
    by_q = {s.question_id: s for s in states}
    wrong_to_review = [
        qid for qid in wrong_qids
        if by_q.get(qid) is None
        or by_q[qid].mastery_level != MasteryLevel.mastered
    ]
    # next practice: <=10 from weak areas, least-recently-practiced, exclude mastered
    weak_qids = {r[0] for r in rows if r[1] is False and _qid_is_weak(r[0], wa)}
    mastered_qids = {s.question_id for s in states if s.mastery_level == MasteryLevel.mastered}
    candidates = [
        (qid, at) for qid, _cor, _tms, at in rows
        if qid in weak_qids and qid not in mastered_qids
    ]
    # distinct question, earliest answered_at
    earliest: dict = {}
    for qid, at in candidates:
        if qid not in earliest or at < earliest[qid]:
            earliest[qid] = at
    ordered = sorted(earliest.items(), key=lambda kv: kv[1])
    next_ids = [qid for qid, _at in ordered[:_MAX_NEXT_PRACTICE]]
    rationale = (
        f"Focus on your weakest domain ({focus.label}) and revisit "
        f"{len(wrong_to_review)} unmastered wrong questions."
        if focus else "No weak areas detected — keep practicing to maintain mastery."
    )
    return ReviewRecommendationOut(
        focus_domain=focus, wrong_to_review=sorted(wrong_to_review),
        next_practice_question_ids=next_ids, rationale=rationale,
    )

from sqlalchemy.orm import Session

    session: Session = Depends(get_session),
    current: CurrentUser = Depends(require_permission("practice:read")),

    return svc.domain_mastery(session, user_id=current.user.id, blueprint=bp)

def get_trend(
    window_days: int = Query(30),
    session: Session = Depends(get_session),
    current: CurrentUser = Depends(require_permission("practice:read")),
):
    if window_days not in (30, 90):
        raise HTTPException(status_code=422, detail="window_days must be 30 or 90")
    return svc.trend(session, user_id=current.user.id, window_days=window_days)

    session: Session = Depends(get_session),

):

    session: Session = Depends(get_session),
    current: CurrentUser = Depends(require_permission("practice:read")),
